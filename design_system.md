# Traveloop Design System

## Design Philosophy
**Modern Minimalism with Travel Warmth**: Clean, spacious interface with warm accent colors that evoke exploration and adventure. The design prioritizes clarity and ease of navigation while maintaining an inviting, friendly aesthetic.

## Color Palette
- **Primary**: Warm Orange (#F97316) - Represents adventure, exploration, and energy
- **Secondary**: Teal (#0891B2) - Represents trust, calm, and discovery
- **Accent**: Amber (#FBBF24) - Highlights and interactive elements
- **Neutral**: Slate grays for text and backgrounds
- **Background**: Clean white (#FFFFFF) for light theme
- **Success**: Emerald (#10B981) - For confirmations and positive actions
- **Warning**: Red (#EF4444) - For alerts and destructive actions

## Typography
- **Display Font**: Poppins (bold, 700) - For headings and hero text
- **Body Font**: Inter (400, 500, 600) - For body text and UI elements
- **Sizes**: 
  - H1: 2.5rem (40px)
  - H2: 1.875rem (30px)
  - H3: 1.5rem (24px)
  - Body: 1rem (16px)
  - Small: 0.875rem (14px)

## Spacing System
- xs: 0.25rem (4px)
- sm: 0.5rem (8px)
- md: 1rem (16px)
- lg: 1.5rem (24px)
- xl: 2rem (32px)
- 2xl: 3rem (48px)

## Component Patterns
- **Cards**: Subtle shadow, light border, rounded corners (8px)
- **Buttons**: Rounded (6px), smooth transitions, hover scale effect
- **Inputs**: Light border, focus ring in primary color
- **Navigation**: Sticky header with clean layout, mobile hamburger menu
- **Icons**: Lucide React, consistent sizing and weight

## Interactive Elements
- Smooth transitions (200-300ms)
- Hover states with subtle color/scale changes
- Loading states with spinners
- Toast notifications for feedback
- Modal dialogs for confirmations

## Responsive Design
- Mobile-first approach
- Breakpoints: sm (640px), md (768px), lg (1024px), xl (1280px)
- Touch-friendly buttons (min 44px height)
- Flexible layouts that adapt to screen size
